Given some sequence of steps, writing a code and getting the output is pretty easy! Want to try a reverse way? If given some input and output, can you find the logic and sequence? Focusing less on concepts, all you need is your logical skill to decipher this "Black Box". Reverse Coding is a logical event which tests your basic logical skills along with pattern analysis and recognition as you play with the code. So, get ready to reverse code in Enkryptiaa '18!

  • There are Three Rounds
    1. Written MCQ test
    2. Logic Round
    3. Decipher and Code Round
  • Logic round consists of Logical questions which can be solved by coding.
  • In the Decipher and coding round, the participants have to write the code for pattern that maps input to output.

  • A maximum of 2 candidates pursuing undergraduate program at their institution are allowed per team.
  • All the participants should bring their receipt and valid I-cards while coming for the contest. A participant without an I-card will NOT be allowed for the contest and the respective team will be disqualified.
  • Marking scheme will be declared on spot.
  • Participants can form teams from different branches/ college/ university/ institute.
  • No two teams must have any common member.
  • The decision of the judges shall be final. Any coaster that violates the rule above or the spirit of the competition will be disqualified
